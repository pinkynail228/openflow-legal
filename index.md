OpenFlow — бесплатный Community beta plugin для Figma Design. Он превращает
текстовое описание в проверяемый preview, а затем создаёт редактируемый user flow
в Figma. Из-за ограниченной серверной квоты generation доступна по revocable
invite token; запросить персональный token может любой пользователь Community.
